# 토닥 (Todak) — 사주 운세 그래프

> 생년월일을 넣으면 내 운세가 곡선으로 보이고, 마지막엔 “너 잘못이 아니야” 한마디를 듣는 무료 사주 웹앱.

단순 텍스트 운세가 아니라, **운(대운·세운·월운)의 흐름을 주가 차트처럼 곡선으로** 보여줍니다.
계산은 전부 브라우저에서 이뤄지고(무료·프라이버시), 마지막엔 자기자비 기반 위로와 안전 안내(109)로 마무리합니다.

기획 철학은 [`기획.md`](./기획.md)를 계승했습니다 — **재미는 사주로, 위로는 진심으로.**

---

## 빠른 시작

```bash
cd C:\tool\port
npm install
npm run dev
# http://localhost:3000 접속
```

빌드/배포:

```bash
npm run build && npm start      # 프로덕션 빌드 확인
# 또는 Vercel 에 그대로 배포 (정적 + 클라이언트 계산이라 서버비 0에 수렴)
```

사주 엔진 정확도 검증(오라클 교차검증):

```bash
npm run verify:saju
```

## 무엇이 있나

- **운세 그래프 (헤드라인)** — 가장 먼저 보입니다.
  - **대운 곡선**: 10년 단위 인생 전체 흐름. 지금 시기를 살구색 점으로 강조.
  - **세운 라인**: 올해부터 10년 연도별 흐름.
  - **월운**: 올해 12개월 흐름(절기 기준).
  - 곡선의 한 점을 누르면 **“왜 이 시기가 이런지”** 설명이 나옵니다. 항상 *참고용* 라벨.
- **사주팔자** — 천간·지지 8자, 오행·십성 표기.
- **오행 밸런스** — 목·화·토·금·수 도넛 + 레이더.
- **사주 성향** — 일간/오행 기반 강점 위주 해석 (겁주는 흉 표현 없음).
- **위로 코어** — “너 잘못이 아니야” 카드(자기자비 3요소) + 1분 호흡 + 한 줄 일기.
- **관상 · 손 모양 (재미, 온디바이스)** — 셀카·손 사진을 MediaPipe로 **기기 안에서만** 읽어 따뜻한 인상으로 풀이. 사진 업로드·저장 없음. 외모 평가·점수 없음. 카메라가 없으면 자가 문답으로 대체. (`/face`)
- **1분 마음 체크 (근거)** — 자기자비척도 단축형(SCS-SF) 12문항 → 자기자비 점수 + 가장 다정함을 늘릴 여지에 CBT식 리프레이밍. 진단 아님. (`/check`)
- **오늘의 나 (통합 리포트)** — 오늘 본 것만 한 장에. **재미(사주·관상)와 근거(마음 체크·위로)를 시각적으로 구분.** 공유 카드 저장. (`/report`)
- **PWA** — 홈 화면에 설치, 한 번 방문하면 오프라인에서도 위로·사주 동작.
- **안전** — 자살예방 상담 **109** 상시 노출, 위기 키워드/낮은 자기자비 감지 시 안내 모달 우선.
- **프로필 여러 개** — 나·가족·친구. 기본은 이 브라우저(localStorage)에만 저장.
- **공유 카드** — 운세 곡선·오늘의 나 이미지로 저장(생년월일·사진 등 개인정보 미포함).

## 사주 엔진 (직접 구현 · 검증됨)

`lib/saju/` 에 자체 구현했습니다. 외부 런타임 의존 없이 브라우저에서 계산됩니다.

| 모듈 | 내용 |
|---|---|
| `solarTerms.ts` | 24절기 — 태양 황경(Meeus 저정밀) 기반 절입 시각 계산(KST) |
| `engine.ts` | 사주팔자(연·월·일·시주), 오행 분포, 십성, 대운/세운/월운 |
| `fortune.ts` | 운세 점수 휴리스틱 (억부용신抑扶用神: 신강/신약 → 용신·기신 → 오행별 유불리) |
| `interpret.ts` | 일간/오행 기반 성향 해석(긍정·중립 톤) |
| `lunar.ts` | 음력→양력 변환(`korean-lunar-calendar`, 오프라인) |

### 정확도 (검증 결과)

`npm run verify:saju` 가 `manseryeok`(2.0.0) 라이브러리를 **오라클**로 삼아 무작위 400건을 교차검증합니다.

- 연주/일주/시주: **100% 일치**
- 월주: **약 99.5%** (불일치는 모두 절입 경계 ±약 11분 이내 출생 — 분 단위 경계 케이스)
- 대운: 방향·간지 **정확히 일치**, 대운수 오차 ≤ 반올림 1

### 한계 (정직하게)

- 절기 시각은 천문 **저정밀** 알고리즘(최대 ~11분 오차)입니다. 절입 직전/직후 분 단위 출생은 월주가 한 칸 달라질 수 있습니다.
- 표준시는 현행 **KST(UTC+9)** 고정입니다. 과거 표준시 변경(127.5°↔135°)·서머타임·진태양시 보정은 적용하지 않았습니다. (필요 시 `manseryeok`의 `trueSolarTime` 옵션처럼 확장 가능)
- 야자시/조자시는 기본 **자정 경계(midnight)** 로 처리합니다.
- **운세 점수는 명리 규칙을 코드로 옮긴 휴리스틱이며 예언이 아닙니다.** 곡선의 오르내림은 “이 시기 오행 기운이 나에게 유리/불리한 쪽인가”의 상대적 표현일 뿐, 길흉의 단정이 아닙니다. 항상 *참고용*.

## 안전 · 윤리 · 프라이버시 (타협 없음)

- 화면 하단에 **자살예방 상담전화 109**(24시간·비밀보장) 상시 노출. `/help` 에 추가 안내.
- 한 줄 일기에서 위기 표현 감지 시, 저장하지 않고 **안내 모달을 먼저** 띄웁니다 (`lib/comfort/messages.ts`의 `detectCrisis`).
- 면책: “이 앱은 의료·심리 치료를 대체하지 않으며, 진단이 아닙니다.”
- 생년월일·일기는 기본적으로 **이 브라우저에만** 저장되고 외부로 전송되지 않습니다.

---

## 무료 · 로컬 전용 (설계 원칙)

토닥은 **로그인·서버·결제 없이** 완전히 동작합니다. 모든 계산은 브라우저에서,
저장은 `localStorage`에만. 생년월일·일기는 외부로 전송되지 않습니다 → 무료로 유지되고,
민감정보가 안전합니다. 정적 + 클라이언트 계산이라 Vercel 무료 티어에 그대로 배포됩니다.

## 기술 스택

Next.js 14(App Router) · TypeScript · Tailwind CSS · Recharts · `korean-lunar-calendar` ·
`@mediapipe/tasks-vision`(온디바이스, 동적 import) · PWA(서비스워커) · Vercel 배포.
검증 전용 devDependency: `manseryeok`, `tsx`.

## 폴더 구조

```
app/            라우트 (/, /app, /face, /check, /report, /help, manifest)
components/     UI (BirthForm, FortuneGraph, FaceReading, PsychTest, ComfortCard, ShareCard, PWARegister ...)
lib/saju/       사주 엔진 (solarTerms, engine, fortune, interpret, lunar, constants)
lib/comfort/    위로 메시지 뱅크 + 위기 감지
lib/psych/      자기자비척도(SCS-SF) 문항·채점·리프레이밍
lib/face/       관상·손 모양 (MediaPipe 로더 + 해석 + 자가 문답)
lib/store/      localStorage (프로필 + 오늘 결과)
public/         sw.js(서비스워커), icon-*.png(PWA)
scripts/        verify-saju.ts(정확도 검증), gen-icons.mjs(PWA 아이콘 생성)
```
