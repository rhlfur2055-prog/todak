## 0. 왜 만드나 (제작자 노트)

이력서를 몇 번이나 고쳐 쓰고, 답장은 더디게 오고, 어느 날 갑자기 현타가 온다.
그 상태에 있는 사람(나를 포함한 지금 세대)이 잠깐 들어와서, 켜자마자 따뜻한 한마디를 듣고,
원하면 재미로 사주·관상도 보고, 한숨 돌리고 나가는 앱.

부수 효과: 이 앱 자체가 좋은 포트폴리오가 된다. 만세력 계산 알고리즘, 브라우저 내 얼굴 분석
(온디바이스 ML), 프론트까지 한 번에 보여줄 수 있다. 취업 준비를 하면서, 취업 준비로 지친
사람을 위로하는 걸 만드는 셈.

---

## 1. 핵심 철학 — "재미는 사주로, 위로는 진심으로"

핵심은 위로다. 사주·관상은 부담 없이 들어오게 하는 가벼운 재미일 뿐이다.

- **위로(본체)**: 빈말("넌 할 수 있어")이 아니라, 심리학에서 검증된 방식(자기자비 3요소)으로 말한다.
- **사주·관상**: 전통이자 재미. "맞힌다"고 우기지 않는다. 항상 "재미로 봐주세요" 라벨.
- **심리검사(선택)**: 근거 기반. 실제 척도(자기자비 등)를 쓰고 출처를 밝힌다. 절대값을 부풀리지 않는다.
- **과장 금지**: 한계는 한계대로 적는다.
- **로그인·광고·결제 없음**: 무료. 켜자마자 위로받는다. 취준생 지갑·시간 안 뺏는다.

---

## 2. 타깃 & 페르소나

- **주 타깃**: 20대 중후반 취업/이직 준비생. 불합격을 반복 중이고, 자존감과 에너지가 낮은 상태.
- **상태**: "내가 문제인가" 자기비난(self-criticism) 루프. 짧게, 가볍게, 부담 없이 위로받고 싶음.
- **사용 맥락**: 밤늦게, 혼자, 폰으로. 길게 안 머문다. 회원가입 시키면 그 자리에서 닫는다.
- **핵심 감정 욕구**: "나만 이런 게 아니구나"(보편적 인간성) + "내 탓이 아니구나"(자기친절).

## 3. 사용자 플로우 (로그인 없이, 1분)

```
진입 → (켜자마자) 오늘의 위로 한마디 카드   ← 이것만으로도 끝나도 됨
  → (원하면) 한 줄 더 보기 / 호흡 1분 / 한 줄 일기 (로컬 저장)
  → (선택·재미) 생년월일 → 사주·운세 그래프
  → (선택·재미) 셀카 → 관상·손금 (사진은 폰에서만, 업로드 안 함)
  → (선택) 1분 심리 체크 → 다정한 코멘트
  → 공유 카드 저장
하단 상시: "지금 많이 힘들다면" → 109(자살예방 상담) 안내
```

각 단계는 **건너뛸 수 있다.** 위로만 받고 나가도 되고, 사주만 보고 나가도 된다.

## 4. 기능 모듈

### 4.0 위로 (본체) — "너 잘못이 아니야"
- 홈을 열면 바로 오늘의 위로 한마디. 들어올 때마다 다르게 로테이션.
- 자기자비 3요소 순서: **인정 → 너만 그런 거 아니야 → 다정함.**
- 실제 문구는 `위로문구.md`(카테고리별 뱅크)에서. 예: "괜찮아, 너 잘못이 아니야",
  "너 혼자 해도 괜찮아", "느려도 괜찮아", "불합격은 네 가치가 아니라 이번 매칭의 문제야".
- 부가: 1분 호흡, 한 줄 일기(localStorage), 공유 카드.
- **빈말 금지**: 압박 대신 인정 → 정상화 → 다정함.

### 4.1 만세력·사주 엔진 (선택·재미)
- **입력**: 생년월일, 태어난 시(모름 옵션), 양력/음력, 성별.
- **출력**: 사주팔자(천간/지지), 오행 분포, 일간, 십성 요약, (확장) 대운/세운.
- **구현**: 오픈소스 JS(`manseryeok`, `korean-lunar-calendar`) → 오프라인·무료.
- **주의**: 한국 표준시 변동·야자시/조자시 처리. 라이브러리 보정 범위를 샘플 생일로 검증.

### 4.2 사주 성향 / 운세 그래프 (선택·재미)
- 일간 + 오행 + 십성 → 강점/기질/"지금 시기" 코멘트(긍정·중립 톤, 겁주는 표현 금지).
- 운세 그래프: 대운(10년) 곡선 + 세운(연도) 라인 + 오행 밸런스. 점수는 명리 휴리스틱(참고용 라벨).
- 상단 고정 라벨: "전통 명리학 기반, 재미로 봐주세요."

### 4.3 관상·손금 (선택·재미, 온디바이스)
- **온디바이스**: MediaPipe Face/Hand Landmarker로 **브라우저 안에서만** 처리. 사진 미전송.
- 항상 따뜻하고 긍정적으로. 외모 평가 아님. 점수·등급 금지. "재미" 라벨.
- 카메라 거부 시 자가 문답으로 대체.

### 4.4 심리검사 (선택·근거)
- 짧고 검증된 척도(자기자비 단축형) → CBT식 다정한 리프레이밍 코멘트.
- 출처·한계 명시("간이 자가검사이며 진단이 아닙니다").

### 4.5 통합 리포트 "오늘의 나" + 공유 카드
- 본 것만 묶기: 위로 한마디(진심) + (있으면) 사주·관상(재미) + (있으면) 심리(근거).
- **재미 vs 근거를 시각적으로 구분.** 공유 카드엔 개인정보(생일/사진) 안 넣음.

## 5. 기술 스택 & 아키텍처

| 영역 | 선택 | 이유 |
|---|---|---|
| 프레임워크 | Next.js(App Router) + TypeScript | 코드마스터에서 쓰던 스택, 익숙함 |
| 스타일 | Tailwind CSS | 빠른 톤 구현 |
| 위로 문구 | 큐레이션 메시지 뱅크(JSON, → 위로문구.md) | API 비용 0, 오프라인 |
| 만세력 | `manseryeok`, `korean-lunar-calendar` | 오픈소스, 오프라인, 무료 |
| 운세 그래프 | Recharts | 반응형 차트 |
| 관상·손금 | MediaPipe Face/Hand Landmarker | 브라우저 내 처리, 사진 업로드 0 |
| 심리검사 | 자체 JSON 척도 + 채점 로직 | 외부 의존 없음 |
| 저장 | localStorage | 로그인·서버 없음 = 무료 + 프라이버시 |
| 배포 | Vercel 무료 / Cloudflare Pages | 정적+클라이언트, 서버비 0 |

**원칙: 로그인 없음 + 모든 계산은 클라이언트(브라우저).** 회원가입·서버·DB 없이 저장은
localStorage만. 생년월일·사진이 서버로 나가지 않는다 → 무료로 유지되고, 민감정보가 안전하다.

개발 중 실행은 `C:\tool\port`에서 `npm run dev` 한 줄.

## 7. 화면 설계 (페이지)

1. **홈** — 켜자마자 오늘의 위로 한마디 + (하단 상시) 위기 상담 안내. 이 화면이 본체.
2. **위로 더 보기** — 다른 문구 / 1분 호흡 / 한 줄 일기.
3. **(선택) 사주·운세 그래프** — 생년월일 입력 → 오행·운세 곡선. "재미로" 라벨.
4. **(선택) 관상·손금** — 카메라/업로드 → 인상 카드. 사진 미전송 배지.
5. **(선택) 심리 체크** — 6~10문항, 1분.
6. **오늘의 나(통합)** — 본 것만 묶음 + 위로 한마디.
7. **공유 카드** — 이미지 저장/공유.
8. **도움받기** — 109/129/청년 마음건강 안내.

## 8. 디자인 톤

- **차분·따뜻·담백.** 슬로건·이모지·그라데이션 남발 금지(AI슬롭 회피).
- 베이스: 아이보리/웜그레이. 포인트는 절제된 따뜻한 한 색(연살구 또는 세이지).
- 큰 여백, 부드러운 라운드, 읽기 좋은 한글. 따뜻함은 색이 아니라 **문장과 여백**으로.
- 카드 한 장에 한 마디. 모션은 잔잔하게, `prefers-reduced-motion` 존중.

## 10. 무료로 유지하는 법

- 로그인·서버 연산 없음(전부 브라우저) → 인증·트래픽 비용 0에 수렴.
- 정적 호스팅 무료 티어(Vercel/Cloudflare Pages).
- v1은 LLM API 미사용(메시지 뱅크) → 변동비 0.
- 데이터 로컬 저장 → DB 비용 0.
- (선택) 나중에 AI 코멘트가 필요하면 그때만 최소 비용 도입. 그 전까진 완전 무료.

## 11. 개발 로드맵

| 단계 | 범위 | 기간(예상) |
|---|---|---|
| **MVP** | 홈 + 위로 카드(위로문구 로테이션) + 호흡/일기 + 안전 안내 | 주말 |
| **v1** | + 사주·운세 그래프 + 공유 카드 | 1~2주 |
| **v2** | + 관상·손금(MediaPipe) + 심리검사 + PWA(설치) + 다국어 | 이후 |

MVP만으로도 "들어와서 위로받고 나가는" 경험은 완성된다. 사주·관상은 그 위에 얹는 재미.
